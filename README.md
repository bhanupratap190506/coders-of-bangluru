# Coders of Bangalore

A Python and Jupyter Notebook based project focused on Bangalore's tech and coding community.

## Project Overview
This project contains developer, startup, AI, Python, and tech community related profile data from Bangalore.

The notebook includes:
- Tech creator profiles
- Startup community pages
- Python & AI communities
- Developer lifestyle pages
- Bangalore tech ecosystem references

## Technologies Used
- Python
- Jupyter Notebook

## Files Included
- coders-of-bangalore.ipynb
- Text data files
- Supporting notebook files

## Purpose
The project is created for learning, data organization, and analysis purposes related to Bangalore's coding ecosystem.
